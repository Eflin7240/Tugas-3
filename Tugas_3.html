Selamat Datang di Website Saya

Halo, nama saya Ahmad Syarif Hidayatullah. Saya seorang mahasiswa yang sedang belajar web development. Website ini saya buat sebagai tugas praktikum untuk mata kuliah Pemrograman Web. Di website ini, saya akan berbagi informasi tentang diri saya, proyek-proyek yang telah saya kerjakan, dan cara menghubungi saya.

About Me

Saya berasal dari Indonesia dan saat ini sedang menempuh pendidikan di sebuah universitas terkemuka. Saya memiliki minat yang besar dalam bidang teknologi dan pengembangan web. Selain belajar, saya juga suka bermain musik dan berolahraga.

My Project

Saya telah mengerjakan beberapa proyek web sederhana sebagai bagian dari pembelajaran saya. Beberapa di antaranya termasuk pembuatan website portofolio, blog pribadi, dan aplikasi web sederhana. Saya selalu berusaha untuk meningkatkan keterampilan saya melalui proyek-proyek ini.
